# SectionLinks

## Properties

<PropsTable of={SectionLinks} />

## Usage

Gives an object of all the registered `Section` for the application by the moment. Each `section` has an `onClick` handler and if it's `selected`.

```javascript
<SectionLinks>
  {({ allLinks }) =>
    Object.entries(([key, value]) => (
      <Item onClick={value.onClick} selected={value.selected}>
        {key}
      </Item>
    ))
  }
</SectionLinks>
```
